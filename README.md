# ◆ 研究
## ◇ 全方位コンテンツの浸透を目指した高臨場感コンテンツデザインシステム
<img src="https://github.com/Akito-ra/markdown_sheet/blob/main/img/design_system_structure.jpg">
<br><br>
<img src="https://github.com/Akito-ra/markdown_sheet/blob/main/img/design_system_overview.jpg">

**研究人数：**
<br>4人
<br> **役割：**
<br>システム設計、クライアントサイド構築
<br> **使用技術：**
<br>html ： UI構築
<br>JavaScript ： コンテンツ制作機能の実装
<br>Python ： データ管理、音源分離、JSONファイル編集
<br>PHP : ログイン情報・セッション管理
<br>その他 : docker ,Herock ,MySQL
<br><br>
**研究背景：**
<br>
近年注目されるようになった全方位コンテンツですが、その認知度は部分的なものとなっています。原因の1つには、全方位コンテンツを扱うには専門知識が必要となり、一般ユーザが気軽に扱うことができないという問題があると考えられます。そこで私たちの研究グループでは、日常生活の中で当たり前に全方位コンテンツが利用される社会の実現を目指して。だれでも手軽に全方位コンテンツに触れられるよう、全方位コンテンツを提示する3次元視聴覚ディスプレイシステムとコンテンツの制作を行うための高臨場感コンテンツデザインシステムを開発してきました。
<br> **システム概要：**
<br>高臨場感コンテンツデザインシステムは、直感的な操作性の実現という理念のもと、全方位動画像や複数の音源データ、テキストデータなどの素材をマウスドラック操作を用いて空間的に配置することで全方位コンテンツの制作を行うことができます。また素材配置だけでなく、音源の編集、バーチャルツアーの作成なども可能となっており、独創性のあるコンテンツの制作が可能となっています。
<br> **ポイント：**
<br>本研究において最も重要となるのが、全方位コンテンツを日常生活の中に浸透させるためにどのようにユーザを動機付け、習慣化させるかという点です。私たちは、「思い出の保存」という用途で全方位コンテンツに触れる機会を増やすことで、全方位コンテンツの認知度の向上を図れると考えました。「負担軽減」や「創作意欲の向上」など学習・解析を繰り返す中で「思い出の保存」という用途で研究システムを利用してもらうために必要な要素を分析、開発を実施しています。
<br><br><br>

* 本作品は、神社の社へ至る参道をきれいに飾ってほしいという依頼を受け、制作した来場者参加型のコンテンツです。
* 参道のh装飾にちなんで、テーマは「願い事」としました。
* 願い事を書いた短冊を専用のボックスに入れることで短冊の色に応じた蓮の花が水面に浮き上がります。花は一定時間経過するか一定数出現したのちに一斉に開花し、散っていきます。

## G-Destroy
* 「VRの楽しさを広げる」、「没入感を向上させる」これら2つの目標のもと作成したコンテンツです
* 本コンテンツ専用のコントローラを作成し、仮想空間と現実空間の親和性を高めること没入感の向上を図っています
<img src="https://github.com/Akito-ra/markdown_sheet/blob/main/img/G_Destroy.jpg" width="80%">

**詳細**
<br>公開時期：2018年10月
<br>対象者　：イベント来場者
<br>制作人数： 9人
<br>制作期間：4ヶ月<br>
<br>
**役割**
<br>ハードウェア開発<br>
**ポイント**
<br>
本コンテンツでは、没入感の向上と酔いの軽減を目的に、仮想空間内のユーザーの傾きに連動して実際に傾く座椅子の開発を試みました。安全面・コスト面の制約から機械的な実装を断念、そこで「ヒトは視線の方向に合わせて無意識に体が傾く」という性質を利用して、台座部分が上下左右前後に伸縮する座椅子を開発しました。
<br>
