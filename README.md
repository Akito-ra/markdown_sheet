# ◆ 研究
## ◇ テーマ：高臨場感コンテンツデザインシステム
**研究概要：**
<br>
VRは現実では行くことができない様々な空間を仮想的に提示できるという利点を持つ一方、専門性やコストの高さから一般ユーザーにはコンテンツを制作・共有することが難しいという課題があります。そこで私は、専門知識を持たないユーザーでもVRコンテンツの制作・共有が手軽に行える3DWebアプリケーションの開発と評価に取り組んでいます。本システムでは、全方位動画像と複数の音声データを利用して、ウェブブラウザ上でVRコンテンツの制作を行うことができます。本研究は、ユーザーによるコンテンツ創作活動を活発化させることに重点を置いており、これまでは直感的な操作性を重視したコンテンツ制作機能の実装やUX向上に必要となる要素の調査・分析を実施、学会やインターロップにて成果報告を行いました。今後は、素材収集を手軽にするための深層学習を用いた音源分離機能や画像識別機能の実装、並びに高齢者の「回想法」の支援などVRの有用性を示すコンテンツの制作を予定しています。

## ◇ Androidアプリケーション
### ■ bonds
* 家事・育児を支援することを目的に開発したアプリケーションです。
* 昨今、スマートフォンの普及に伴う子供のコミュニケーション能力低下が問題視されています。親が家事に集中するために子供にスマートフォンを渡してしまうことが原因の1つとされています。そこで、親の家事を支援しつつ親子のコミュニケーションを促進するアプリケーションの開発を行いました。
* このアプリケーションでは、家事の手伝いなど子供が良いことをすることでポイントが貯まります。子供は、貯ったポイントを使ってキャラクターを模った仮想的な動くシールを入手することができます。<br>

**詳細**
<br>対象者　：一般家庭の親子
<br>制作人数： 1人
<br>制作期間：1ヶ月<br>
<br>
**使用した技術**
<br>AndroidStudio, Java：アプリケーションの実装
<br>
**役割**
<br>企画立案、アプリケーション実装
<br>
**ポイント**
<br>
本アプリケーションでは、入手したシールを利用してレイアウトを作成することができます。レイアウトの作成や家事の手伝いを通して親子のコミュニケーションの促進を図ります。「シールを張る」と「親子の絆を取り戻す」という2つ意味を込めて「bonds」という名前を付けました。また本アプリケーションは、子供の落書き防止や想像力の発達にもつながると考えています。<br>

# ◆ ハッカソン
**テーマ**
<br>
「ITの発展に伴う銀行の新たなサービス」
<br>
**提案したサービス**
<br>
「軽融資サービス」
<br>
**概要**
<br>
軽融資サービスは、IT技術の発展に伴い生まれてきた新たなお金の流れに対応したい金融機関と小規模事業を始めたい一般ユーザーを対象としており、事業主を支援するための金融機関ならではのプログラムが多数盛り込まれています。<br>
本サービスの最大の特徴は、金融機関が貸付を行わないという点です。資金調達にはクラウドファンディングを利用、金融機関がそれまでに培ってきた信頼を貸し出して事業主の後見人となることで、支援者・被支援者ともに安心して取引を行える環境を提供します。
<br>
**提案システム**
<br>
本サービスでは、AIを利用することで事業主の審査工程の効率化を図ります。本ハッカソンでは、デモとしてChatbotとGithubを連携させ、IDを入力することで自動的にスクレイピングが行われるシステムの構築を行いました。
<br>
